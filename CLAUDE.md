## Project

**BooksWithYou** — a social reading tracker. Members create reading sessions for books, others join and log chapter progress, with a flat discussion thread and emoji reactions per session.

- **Backend:** Supabase (Postgres, Auth, Realtime, Storage)
- **Web:** Next.js (React, TypeScript, Tailwind CSS)
- **Mobile:** React Native (Expo)
- **Spec:** See `SPEC.md` for full details, schema, and API routes.

## Getting Started

Project not yet initialized. Start with:
